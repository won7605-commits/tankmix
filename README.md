# 농약 혼용정보 검색 — 웹 배포

이 폴더는 **GitHub Pages 배포 전용**입니다. 여기 `index.html`이 웹사이트로 서비스됩니다.

- 데이터/빌드 작업은 `OneDrive\바탕 화면\클로드폴더\혼용정보앱` 폴더에서 합니다.
- 그 폴더에서 **업데이트.bat** 실행 → 이 폴더의 `index.html`이 자동 갱신됩니다.
- 그 다음 **GitHub Desktop에서 Commit → Push** 하면 웹사이트가 최신본으로 바뀝니다.
- `robots.txt` 는 검색엔진 노출 차단용입니다 (URL 아는 사람만 접근).

## 업데이트 순서 요약
1. 엑셀(원본) 수정 후 저장·닫기
2. `혼용정보앱\업데이트.bat` 더블클릭
3. GitHub Desktop → Commit → Push
4. 1~2분 후 웹 URL에 반영
